【客户使用 — 3 步】
1. 安装 APK，授予麦克风权限
2. 打开 App → 开「自动对话」→ 点麦克风说话（WiFi 或 4G 均可）
3. 无需填 Proxy、无需开电脑；对话自动保存在手机本地

【部署说明（我方一次性配置）】
1. cd worker → wrangler secret put OPENAI_API_KEY → wrangler deploy
2. GitHub Secrets 设置 VITE_API_BASE_URL = https://voice-ai-proxy.boonshee-ai.workers.dev
3. push 后 Actions 自动打出预置云端地址的 APK
